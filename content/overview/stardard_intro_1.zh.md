# 医学影像检查项目名称及编码标准

*Standard for the Naming and Coding of Medical Imaging Examination Items*

## 1 范围

本标准规定了医学影像检查项目的名称及分类编码的规则。

本标准适用于医疗卫生机构开展医学影像检查服务，进行卫生健康信息统计与分析，医学影像相关软硬件开发。

## 3 术语和定义

下列术语和定义适用于本文件。

### 3.1 医学影像

因医疗工作需要，借助于某种介质（如X线、电磁场、超声波等）与人体相互作用，把人体内部组织器官结构、密度、组织成分等信息以影像方式表现出来的图像。

### 3.2 医学影像检查项目名称

为科学反映医学影像检查项目的用途、操作过程等特性，对检查项目的规范命名。

### 3.3 医学影像检查项目编码

医学影像检查项目的唯一标识码，由数字、字母等符号组成。

### 3.4 编码规则

对数据进行编码时所依据的规范或原则。

### 3.5 模态

用于创建体内图像的不同类型的医学成像技术。

### 3.6 设备类型

同一种医学成像技术下，医学影像设备的不同细分类型，例如16排CT和64排CT、1.5TMR和3.0TMR等。

### 3.7 检查部位

在进行医学检查时，需要进行观察、诊断、治疗或操作的特定身体区域。

### 3.8 扫描方式

对多种模态和类型的医学影像设备进行成像检查操作的总称。

### 3.9 后处理方式

医学影像后处理是指对医学影像进行数字处理以改善图像质量、增强图像信息、辅助诊断、或者进行其他分析和处理的过程。这些处理可以包括图像增强、去噪、对比度调整、3D重建等技术。

### 3.10 体位

患者在进行X线拍摄时的姿势或位置，以确保获得特定部位的最佳影像。

### 3.11 附加码

附加码是为标准检查项目扩展预留的编码。

## 4 缩略语

下列缩略语适用于本文件。

* CT（computed tomography）计算机体层摄影术
* MRI（magnetic resonance imaging）磁共振成像术（在本标准中，模态统一用 MR 来表示 MRI 磁共振成像术）
* DR（digital radiography）数字放射影像
* CR（computed radiography）计算机 X 线摄影术
* MG（mammography）乳房 X 线照相术
* RF（radio fluoroscopy）放射荧光透视（数字胃肠）

## 5 总则

### 5.1 唯一性

每个医学影像检查项目名称及编码对应的数据采集内容在适用范围上具有唯一性、不可替代性。同一检查项目的检查目的、扫描范围、扫描内容、扫描方式不可以有重复（多个不同）的名字，其解决的临床适用性是唯一的。检查项目可以再次细分或者被包含，但不可以被替代。

### 5.2 全面性

检查项目应全面覆盖常规临床应用，同时保留对新兴技术和罕见应用的开放性。

### 5.3 明确性与准确性

检查项目应清楚、明确，避免含糊或歧义，确保所有使用者都能准确理解和遵循。

### 5.4 可拓展性

检查项目名称及编码的框架具有扩展空间，能根据发展情况，对检查项目名称及编码进行增减和调整。

### 5.5 可组合性

检查项目编码能支持医疗机构根据需要针对某疾病、某类疾病进行多个检查项目组合，表现形式是某专病组合项目。

## 6 医学影像检查项目名称及编码

### 6.1 医学影像检查项目分类

医学影像检查项目应按照检查部位、检查类型、特殊要求等分为不同的类别，以便管理和使用。

#### 6.1.1 按检查部位分类

检查部位的定义和划分是本标准的核心组成部分，CT、MR、DR 三种模态共享一套部位分类体系，并采用统一的编码。该部位体系分为三级分类，包括一级部位、二级部位和三级部位。这里的部位是广义的分类概念，包含了人体解剖部位、器官，同时也可能包含系统或其他的分类。

一级部位：是根据人体基本解剖结构和医学影像检查工作实际需求划分，包括头部、颈部、脊柱、胸部、腹部、盆部、四肢及关节。

由于血管系统在医学影像成像技术中的重要性和其在多种疾病诊断中的关键作用，因此将血管列入一级部位分类。此外，为了满足临床的灵活性需求，增加了"非指定部位"分类。

医学影像一级部位分类及编码，见表1。

| 一级编码 | 一级部位  |
| ---- | ----- |
| 01   | 头部    |
| 02   | 颈部    |
| 03   | 脊柱    |
| 04   | 胸部    |
| 05   | 腹部    |
| 06   | 盆部    |
| 07   | 四肢及关节 |
| 08   | 血管    |
| 99   | 非指定部位 |

二级部位：是从一级部位分类按影像检查特点划分出的特定区域或人体结构，是基本的扫描操作单元和观察诊断单元，也是数据互联互通和统计分析重要单元。

三级部位：是对二级部位更精细的划分，主要目的是明确具体的检查部位或器官，满足具体的、明确的或者特定的临床检查和诊断需求。

#### 6.1.2 按模态和设备类型分类

本标准中，主要收录了 CT、MR、X 线检查三种模态类型。由于不同设备类型的检查费用存在差异，因此项目名称和编码中加入了设备类型编码。本标准推荐采用不区分具体设备的检查项目和编码。

#### 6.1.3 按扫描方式分类

在不同的医学影像模态中，所采用的扫描方式存在一定的差异，并且名称也各不相同。例如，在 CT 和 MR 中，通常将扫描方式称为"扫描方法"或"成像技术"，而在 X 线中则称为"摄影技术"。为了统一分类，本标准不再区分不同的医学影像模态，而是将它们统一称为"扫描方式"。

为了便于理解和应用，本标准对医学影像模态中的扫描方式进行如下分类，而这些大类又可以进一步细分，详细扫描方式分类及编码表见附录 D。

CT 扫描方式：包括平扫、增强、血管成像、特殊扫描等。

MR 扫描方式：包括平扫、增强、血管成像、水成像、功能成像和特殊成像等。

X 线扫描方式：包括常规 X 线摄影、X 线透视、乳腺 X 线、X 线造影等。

#### 6.1.4 按后处理方式分类

医学影像数据的后处理方法是指对原始影像数据进行计算机计算处理的所有方式。本标准将这些后处理方法分为三维重建、定量测定、后处理仿真和人工智能分析后处理等类别。这些类别可以进一步细分，详细的后处理分类和编码表见附录 E。

#### 6.1.5 按体位分类

在医学影像技术领域中，X 线摄影的应用范围最为广泛，其体位亦具有多样性。这些体位均旨在更好地展示特定部位的结构或病变，因此需要检查中采用不同的体位。

例如正位、侧位、斜位、轴位等。体位分类及编码表见附录 F。

### 6.2 医学影像检查项目名称

医学影像检查项目名称用于临床医生开具检查申请单、患者支付检查项目费用、医学影像技师选择特定的操作流程和技术参数、影像医生给出相应的诊断报告等场景、以及后期数据统计和数据分析，因此，项目名称应具备以下特点：科学准确、简洁清晰、唯一性、可读性、可组合、可计算机识别，以保证其适用于各类场景。

本标准中的检查项目名称均以“\*”开头，用于实际标准上线时在系统中与非标准项目进行区分，可根据实际需要取消。

本标准的格式按照内容组合，可根据实际需要采用不同的组合方式，但编码应保持统一。

#### 6.2.1 项目名称的构成

设备模态及类型：例如 CT、16 排 CT、MR、3.0TMR、DR、MG 等。

检查部位：例如颅脑、胸部、上腹部等。

扫描方式：例如平扫、增强等。

其他特定信息：例如后处理方式、体位等。

#### 6.2.2 CT / MR检查项目名称格式

\*设备模态及类型+检查部位+（扫描方式+后处理）

设备类型：检查项目直接用 CT、MR、DR 来表示设备类型，说明该检查项目不区分具体的设备类型，但在一些地区医保收费规则中，不同设备类型有不同的收费标准，因此需要将“16 排 CT、64 排 CT、1.5TMR、3.0TMR”设备类型信息编入项目名称。

检查部位：指检查项目的扫描部位，是最细分检查部位，即三级部位。

扫描方式及后处理采用英文括号进行组合。

示例：

```
CT 颅脑（平扫 + 三维重建）
MR 胸部（平扫 + 增强）
3.0TMR 胸部（平扫 + 增强）
```

#### 6.2.3 X线检查项目名称格式：

\*设备类型+检查部位+扫描方式+体位

其中 DR 检查的扫描方式默认为 X 线摄影，因此不在名称中表达。

示例：

```
DR 颈椎正位
MG 双侧乳腺钼靶正、侧位
```

### 6.3 医学影像检查项目编码

#### 6.3.1 编码方法

医学影像检查项目编码按 GB / T 7027 规定的方法。

#### 6.3.2 编码规则

编码采用数字字母混合码，同类目位数、类型一致。

各级类目中的"非指定部位"类编码定为 99。

每个编码都有其特定的含义和用途，可根据需要进行组合和扩展。

字母"i、o、l"容易与数字"1、0"在视觉上混淆，不纳入编码。

编码格式可根据需要进行调整。

#### 6.3.3 编码元素

医学影像检查项目编码采用 14 位码，码位结构包括 5 部分，分别对应不同的信息元素，见表 3。
```
第 1 - 2 位为模态及设备类型编码；2x 表示 CT 检查，其中 x 表示不同设备类型，如 20 表示不区分设备的 CT 检查；3x 表示 MR，其中 30 表示不区分设备的 MR 检查；4x 表示 X 线检查，其中 40 表示 DR、44 表示 MG、45 表示 RF 等。

第 3 - 8 位为部位编码，其中第 3 - 4 位为一级检查部位编码；第 5 - 6 位为二级检查部位编码；第 7 - 8 位为三级检查部位编码。

第 9 - 10 位为扫描方式编码。

第 11 - 12 位为后处理 / 体位编码，对于 CT / MR 的检查项目，此处为后处理编码；而对于 X 线检查项目，则为体位编码。

第 13 - 14 位为附加码，附加码是为标准检查项目扩展预留的编码。

```

| **码位**   | 第1-2位 | 第3-4位 | 第5-6位 | 第7-8位 | 第9-10位 | 第11-12位  | 第13-14位 |
| -------- | ----- | ----- | ----- | ----- | ------ | -------- | ------- |

示例：

```

20010701111100
20 01 07 01 11 11 00

```

### 7、个性化检查项目的名称及编码管理方案

在某些医疗机构中，由于专业特色或特定检查项目的开展，仅靠现有的标准检查项目可能无法满足全面开单需求。因此，为了更好地满足实际需要，医疗机构可在标准编码体系基础上增加附加信息，对个性化的检查项目进行扩展编码和命名。

示例：

机构为明确脊柱锥体位置和数量，可在标准项目名称“\* CT 颈椎（平扫）”后，用“-”连接扩展信息。

* \*CT 颈椎（平扫）-（C1-C3），编码为 20030202110000-x
* \*CT 颈椎（平扫）-（C2-C4），编码为 20030202110000-x

在 MR 检查中，为明确项目包含的特定扫描序列，可在标准项目名称后添加扩展信息进行定义。

* \*MR 颅脑（平扫）-含 DTI，编码为 30010100110000-x

为了方便临床医生开单，可以将多个标准项目组合成一个个性化的检查项目。

* \*MR 颅脑（平扫）-癫痫序列，编码为 30010100110000-x

这些名称在实际工作中，可以采用别名以方便使用，但编码需保持一致。
